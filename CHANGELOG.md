# 更新日志

## 2.0.6

🎉 **heybox Lite 2.0.6 更新**

【新功能】
- 📚 新增阅读中心：整合继续阅读、阅读时长、稍后看和历史记录，历史记录可与小黑盒账号同步
- 👤 新增我的动态：可以查看自己发布的动态和投稿
- 📝 新增评论草稿：未发送的评论和回复会自动保留，下次打开可以继续编辑
- 🌐 新增网络模式：可以在省流量、标准和原图之间切换
- 🩺 新增一键自检：可以检查登录、网络、缓存和更新服务，反馈问题更方便

【体验优化】
- 🔙 修复阅读中心返回层级，从稍后看、历史记录和阅读时长返回时不再直接跳到“我的”
- 💬 修复评论以官方表情开头时后续文字消失、只显示一个表情的问题
- 🖼️ 评论支持显示多张图片，图片布局更加紧凑
- 🎞️ 评论中的 GIF 动图现在可以正常识别和播放
- 🧩 优化评论富文本解析，减少文字缺失、乱码和异常内容
- 🛡️ 减少不必要的接口重试，降低触发风控的概率，同时保持点赞、收藏、关注和评论功能可用

【其他】
- 📅 每日签到仍暂时关闭，避免不稳定请求影响其他功能
- 🛠️ 优化搜索、信息流、图片手势和页面状态管理，提升稳定性

## 2.0.5

🎉 **heybox Lite 2.0.5 更新**

【新功能】
- 📖 阅读时长：在「我的」里新增阅读时长，能看到今天和累计看了多久、文章与帖子各占多少、最近 7 天的阅读趋势，还有你最常逛的社区排行
- 💬 评论区补齐官方标识：**置顶评论**、楼主的「作者」标、用户**等级**、带 **Cy** 标识的评论，现在都能看到了；楼中楼（二级评论）在原有连续排版的基础上，也用上了这些标识，并和文字、冒号对齐
- 🏷️ 帖子话题标签：帖子和文章下方会显示它属于哪个社区（话题），和官方一样
- 🔍 放大后也能左右滑：多图帖的大图查看界面现在可以左右滑动看上一张 / 下一张，右上角显示「1/3」这样的页码；图片放大时手势不冲突（原来只能在帖子里滑，进大图后只能一张张点）
- 💾 存图与查看原图：大图界面可以直接保存到相册；「查看原图」会顺便告诉你原图多大（比如 379KB），流量党可以看情况再点
- 📋 评论长按复制：看到想抄的评论，长按就能复制
- 🔙 退出确认：可以在设置里开启，开了以后在主页要连按两次返回才退出，防止手滑退出去

【体验优化】
- 🌊 动画与转场重做：社区和「我的」之间改成整页左右平移切换，更跟手；切换页面、从帖子 / 搜索 / 个人页返回时，都会停在你原来看的位置，不再跳回顶部
- 🔘 帖子详情操作区：底部拆成点赞、收藏、缓存、评论四个独立按钮（原来的「稍后看」改名「缓存」），图标也换成了官方那套
- 💬 评论区更顺手：「热门 / 最新」点一下直接切换，不用再弹菜单去选；点赞数很大时自动缩写（比如 1.2万），图标和数字排得更紧凑
- 🎨 「我的」与设置页再打磨：图标统一成清爽的单色小方块，开关改成滑动的样式，分组和留白看着更舒服
- 🖼️ 图片查看器：下载按钮改成不遮挡画面的小图标，多图切换和原图加载更顺滑

【其他】
- 📱 优化了在较老 Android 系统和低配手表上的兼容与显示
- 🛡️ 在线状态改为每 10 分钟更新一次；头像和公开资料只在第一次登记时上传，之后只发盒子 ID、版本号和累计阅读时长，不携带 Cookie、登录凭据或浏览内容

## 2.0.4

🎉 **heybox Lite 2.0.4 更新**

【新功能】
- 🖼️ 图片离线缓存：浏览过的图片会自动存到本地，再次查看秒开、离线也能看、更省流量；缓存用量可在「设置」查看，支持自动清理
- 🎞️ 动图播放：帖子里的 GIF 动图现在能正常播放（详情页），低配手表会自动降级保护，不卡顿
- 🔎 搜索升级：修复了「只能搜到十来条」的问题，现在支持持续翻页、不断加载更多结果

【体验优化】
- 📰 文章排版对齐官方：新增引用块、图片下方的说明文字（图注），标题层级、发布时间与归属地显示更完整
- 🖼️ 多图帖改为左右滑动图集，浏览更顺手（和官方一致）
- 💬 楼中楼评论修复：回复里的用户名不再被截断，改成与官方一样的连续排版
- 🔙 从帖子返回搜索结果时，列表停在原来的位置，不再跳回顶部
- 🎨 「我的」与设置页整体重构：线性图标 + 卡片式布局，风格更统一，并修正了部分按钮配色

【其他】
- 🗓️ 每日签到暂时关闭（受官方风控限制，第三方客户端暂时无法稳定通过校验）
- 🛠️ 修复若干稳定性问题，提升整体流畅度
