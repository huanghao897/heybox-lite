# 更新日志

## 2.0.5

### 新增

- **阅读时长**：在「我的」里新增阅读时长，可以看到今天和累计的阅读时间、文章与帖子的占比、近 7 天的阅读趋势，以及最常看的社区。
- **评论区官方标识**：评论区现在会显示置顶评论、作者、用户等级和 Cy 评论标识，一级评论和楼中楼的排版也向官方看齐。
- **帖子话题标签**：帖子和文章下方会显示所属社区的话题标签。
- **多图左右滑动**：多张图片的帖子点开大图后，可以左右滑动查看上一张 / 下一张，右上角显示当前页码（文章里的长图仍保留双击缩放）。
- **保存图片与查看原图**：图片查看器新增保存到相册，「查看原图」会显示原图大小。
- **退出确认**：新增开关，开启后在主页需要连续返回两次才会退出应用。
- **评论长按复制**：长按评论即可复制内容。

### 优化

- **动画与转场**：重做了页面进出和滑动返回的动画，社区与「我的」之间改为整页左右平移切换；从帖子、搜索结果或个人页返回时，会保留原来的浏览位置，不再跳回顶部。
- **帖子详情**：底部操作区拆成点赞、收藏、缓存、评论四个独立按钮（原「稍后看」改名为「缓存」）；点赞、收藏、评论均改用与官方一致的图标。
- **评论区**：点一下即可在「热门 / 最新」之间切换；点赞数较大时自动缩写，图标与数字排列更紧凑。
- **设置与「我的」页面**：重新设计，图标统一为单色芯片，设置开关改为滑块，分组和留白更清爽。
- **图片查看器**：下载改为不遮挡画面的小按钮，多图切换和原图加载更顺滑。

### 稳定性

- 改善了在较低版本 Android 系统和低配置设备上的兼容性与显示效果。
- 在线状态改为每 10 分钟更新一次；头像和公开账号资料只在首次登记时上传，之后仅更新黑盒 ID、版本号和累计阅读时长，不携带 Cookie、登录凭据或浏览内容。
